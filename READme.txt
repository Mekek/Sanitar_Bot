файл main.py - главный файл с кодом
Sanitar_bot_project.pptx - презентация проекта
пояснительная записка.docx - пояснительная записка к проекту

Для запуска бота необходимо:
1) открыть файл main.py
2) через терминал выполнить следующие команды:
	pip install python-telegram-bot
	pip install requests
3) открыть телеграм и найти через поиск sanitar2021_bot
4) ввести в чат с ботом команду /start
5) чтобы посмотреть остальные основные команды, нужно ввести команду /help
